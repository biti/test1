# Tank Battle Infinite

一个使用 Python 与 pygame 编写的简易坦克大战原型。玩家控制坦克在随机生成的关卡中与敌方坦克作战。每当清除当前关卡的敌人后，会自动进入下一关并生成全新的随机地图，理论上可以无限闯关。

## 安装依赖

```bash
pip install pygame
```

## 运行

```bash
python game.py
```

## 操作说明

- **方向键 / WASD**：控制坦克移动与旋转
- **空格键**：发射炮弹
- **Esc**：退出游戏

玩家被击毁时会从第一关重新开始。随着关卡数提升，敌方坦克数量会逐渐增加。
